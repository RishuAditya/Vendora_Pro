# 🛒 Vendora Pro - Multi-Vendor Marketplace

Vendora Pro is an enterprise-level multi-vendor e-commerce platform built with **Python (Flask)** and **MySQL**. It features advanced analytics, role-based access control, and a modern **Glassmorphism UI**.

## ✨ Key Features

- **3 Role System:** Admin, Seller, and Customer.
- **Security:** Password hashing using **Bcrypt**.
- **Seller Intelligence:** Automated performance scores and badges.
- **Admin Control Tower:** Full control over sellers, commissions, and revenue.
- **Responsive UI:** Built with Bootstrap 5 and Custom CSS.

## Tech Stack

- **Backend:** Flask, Flask-SQLAlchemy, Flask-Login, Flask-Bcrypt
- **Database:** MySQL
- **Frontend:** HTML5, CSS3 (Glassmorphism), Bootstrap 5, JS
- **Environment:** python-dotenv

## Installation & Setup

1. Clone the repository: `git clone https://github.com/RishuAditya/Vendora_Pro.git`
2. Create virtual environment: `python -m venv venv`
3. Activate environment: `venv\Scripts\activate` (Windows)
4. Install requirements: `pip install -r requirements.txt`
5. Set up `.env` file with your DB credentials.
6. Run application: `python run.py`
